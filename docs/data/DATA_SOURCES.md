# 数据来源与版权说明

本文档说明项目中使用的历史书籍数据来源、版权状态以及推荐的历史书籍资源。

## 版权状态

### 《史记》版权状态

**《史记》已进入公共领域（Public Domain）**，可以自由使用：

- **成书时间**：西汉时期（约公元前 91 年）
- **作者**：司马迁（已去世超过 2000 年）
- **版权状态**：✅ **完全免费，无版权限制**
- **使用范围**：可以自由复制、分发、修改、商业使用

**注意**：
- 虽然《史记》原文无版权，但**现代白话文翻译版本**可能仍有版权
- 建议使用**原文版本**或明确标注为公共领域的版本
- 如果使用现代学者的注释版本，需要确认版权状态

- **网址**：https://zh.wikisource.org

## 数据准备建议

### 1. 优先使用原文版本
- 使用古籍原文，避免使用现代翻译版本（可能有版权）
- 原文版本更符合学术规范

### 2. 数据格式
- 建议从维基文库或 CTEXT 获取，格式规范
- 可以导出为 Markdown 或纯文本格式
- 保持章节和段落结构

### 3. 数据质量检查
- 检查文本完整性（是否有缺漏）
- 检查编码问题（确保 UTF-8）
- 检查标点符号（古籍可能使用不同标点）

### 4. 数据预处理
- 统一段落格式
- 添加段落编号（便于引用）
- 标记章节标题
- 提取关键信息（人物、地点、事件）

## 版权注意事项

### ✅ 可以自由使用
- 古籍原文（已进入公共领域）
- 古籍的现代标点整理版本（如果明确标注为公共领域）
- 古籍的数字化版本（如果来源明确为公共领域）

### ⚠️ 需要注意
- 现代学者的注释版本（可能有版权）
- 现代白话文翻译版本（可能有版权）
- 现代学者的研究著作（有版权）
- 古籍的现代排版设计（可能有版权）

### 📝 建议做法
1. **明确标注数据来源**：在项目中标注文本来源（如"来源：维基文库"）
2. **使用原文版本**：优先使用古籍原文，避免版权问题
3. **保留原始信息**：保留章节、段落等原始结构信息
4. **注明版权状态**：在项目文档中说明使用的文本版权状态

## 数据获取示例
