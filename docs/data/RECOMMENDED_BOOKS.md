# 推荐书籍：秦汉时期（聚焦西汉）

本文档推荐 5 本聚焦秦汉两个朝代（尤其是西汉）的经典史书，用于本项目的数据准备。

## 推荐书籍列表

### 1. 《史记》（西汉·司马迁）

**推荐理由**：
- ✅ **必选**：西汉时期写成，作者司马迁生活在汉武帝时代，对西汉历史有第一手资料
- ✅ 包含大量秦汉历史：从秦朝到汉武帝时期
- ✅ 纪传体通史，人物传记丰富，适合提取人物关系
- ✅ 已进入公共领域，无版权问题

**重点章节**（秦汉相关）：
- 《秦始皇本纪》- 秦朝历史
- 《项羽本纪》- 楚汉之争
- 《高祖本纪》- 汉高祖刘邦
- 《吕太后本纪》- 吕后时期
- 《孝文本纪》《孝景本纪》《孝武本纪》- 西汉各帝
- 《陈涉世家》《萧相国世家》《曹相国世家》等 - 重要人物
- 《淮阴侯列传》《留侯世家》等 - 韩信、张良等

**获取渠道**：
- 维基文库：https://zh.wikisource.org/wiki/史記
- CTEXT：https://ctext.org/shiji

---

### 2. 《汉书》（东汉·班固）

**推荐理由**：
- ✅ **必选**：第一部断代史，专门记载西汉历史（从汉高祖到王莽）
- ✅ 内容详实，补充《史记》未覆盖的西汉中后期历史
- ✅ 与《史记》互补，可以交叉验证
- ✅ 已进入公共领域

**重点内容**：
- 《高帝纪》- 汉高祖
- 《惠帝纪》《高后纪》- 惠帝、吕后
- 《文帝纪》《景帝纪》《武帝纪》等 - 西汉各帝
- 《萧何传》《曹参传》《张良传》《韩信传》等 - 重要人物传记
- 《地理志》《食货志》- 地理、经济信息

**获取渠道**：
- 维基文库：https://zh.wikisource.org/wiki/漢書
- CTEXT：https://ctext.org/han-shu

- ✅ 时间线清晰，便于提取事件和地理信息
- ✅ 涵盖秦朝到西汉的完整时间线
- ✅ 已进入公共领域

**重点卷次**（秦汉相关）：
- 卷六-卷七：秦朝（秦始皇、秦二世）

## 书籍选择说明

### 为什么选择这 5 本书？

1. **时间覆盖完整**：
   - 《战国策》→ 战国时期（秦统一前）
   - 《史记》→ 秦朝 + 西汉前期
   - 《汉书》→ 西汉完整历史
   - 《后汉书》→ 东汉历史
   - 《资治通鉴》→ 时间线梳理

2. **体裁互补**：
   - **纪传体**：《史记》《汉书》《后汉书》- 以人物为中心
   - **编年体**：《资治通鉴》- 以时间为线索
   - **国别体**：《战国策》- 以国家为单位

3. **数据提取优势**：
   - **人物数据**：纪传体书籍人物信息丰富
   - **关系数据**：人物传记中关系描述详细
   - **事件数据**：编年体时间线清晰
   - **地理数据**：各书都有地理信息

### 优先级建议

**第一阶段（MVP）**：
1. 《史记》- 必选，最核心
2. 《汉书》- 必选，补充西汉中后期

**第二阶段（扩展）**：
3. 《资治通鉴》- 时间线补充
4. 《后汉书》- 东汉历史
5. 《战国策》- 背景补充

## 数据提取建议

### 1. 提取顺序

建议按以下顺序提取数据：

1. **《史记》** → 提取人物、关系、地点、事件
2. **《汉书》** → 补充人物、关系，验证《史记》数据
3. **《资治通鉴》** → 提取事件时间线，补充地理信息
4. **《后汉书》** → 提取东汉人物和关系（可选）
5. **《战国策》** → 提取战国背景人物（可选）

### 2. 提取重点

#### 人物提取重点
- **《史记》**：刘邦、项羽、韩信、张良、萧何、曹参、陈平、周勃等
- **《汉书》**：补充西汉中后期人物（如霍光、张骞等）
- **《资治通鉴》**：补充时间线中的人物

#### 关系提取重点
- 楚汉之争时期的关系网络
- 西汉开国功臣之间的关系
- 皇室与大臣的关系
- 敌对关系（如刘邦 vs 项羽）

#### 地点提取重点
- 重要城池：咸阳、长安、洛阳、彭城等
- 战场：巨鹿、垓下、鸿门等
- 地理区域：关中、关东、巴蜀等

#### 事件提取重点
- 秦朝统一六国
- 陈胜吴广起义
- 楚汉之争
- 西汉建立
- 重要战役和政治事件

### 3. 数据验证

- **交叉验证**：同一人物/事件在《史记》和《汉书》中的描述对比
- **时间验证**：使用《资治通鉴》的时间线验证事件时间
- **地理验证**：对比不同书籍中的地理描述

## 获取和使用建议

### 1. 文本获取

1. 从维基文库或 CTEXT 下载原文
2. 保存为 UTF-8 编码的文本文件
3. 保持章节结构

### 2. 数据预处理

1. **章节划分**：按原书章节结构划分
2. **段落编号**：为每个段落添加编号（便于引用）
3. **标注来源**：每个段落标注来源书籍和章节

### 3. LLM 提取

使用项目中的 `scripts/extract_with_llm.py` 脚本：
- 先提取人物
- 再提取关系（需要人物列表）
- 然后提取地点和事件

### 4. 人工 Review

- 使用管理后台的 Review 工具审核
- 交叉验证不同书籍的数据
- 修正 LLM 提取的错误
